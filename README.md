# Orange blog - the social network
This is a site for writing posts. Here users(authors) can create posts, leave comments and likes.


## Capabilities:

1. Creation of posts
2. Leaving comments
3. Liking posts
4. Registation of user
5. LogIn/LogOut
6. Changing of profile setting
7. Following of users
